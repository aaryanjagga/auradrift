Aura Drift 

A classic arcade experience evolved for the modern web. Aura Drift is a fully responsive, browser-based game built entirely with the "Trio" of web technologies: HTML5, CSS3, and Vanilla JavaScript.

No libraries. No frameworks. Just pure logic.
